## 📣 프로젝트 소개

## 🛠️ 주요 기능

### ✅ 회원가입

이메일 인증, 인증번호 확인, 닉네임 중복 확인, 비밀번호 형식 확인, 프로필 이미지 크기 확인 절차를 거쳐서 회원가입을 진행할 수 있어요.<br/>

<img src="https://github.com/CHZZK-Study/Balance-Talk-Frontend/assets/84820008/fbc1af36-81bc-4fe0-a4c1-ed8272b3e2c6.png" width="700" height="363"/>


### ✅ 로그인
로그인을 통해 회원이 사용할 수 있는 여러 기능들을 사용할 수 있어요.<br/>

<img src="https://github.com/CHZZK-Study/Balance-Talk-Frontend/assets/84820008/3fd0cbd3-259b-44b1-b675-10796153268b.png" width="700" height="363"/>

### ✅ 비밀번호 찾기
비밀번호를 잊었다면 비밀번호 찾기를 통해 임시 비밀번호를 발급받을 수 있어요.<br/>

<img src="https://github.com/CHZZK-Study/Balance-Talk-Frontend/assets/84820008/ff3cb250-bc3f-4f89-bccd-e7f77750ccc5.png" width="700" height="363"/>

### ✅ 마이페이지

마이페이지에서는 활동내역 조회, 회원정보 수정, 회원 탈퇴 기능을 제공하고 있어요.<br/> 

<img src="https://github.com/CHZZK-Study/Balance-Talk-Frontend/assets/84820008/9ab0b4fb-6fd5-4293-8e9a-e7d243e02c74.png" width="700" height="363"/>

활동내역은 크게 내가 작성한 게시글 조회 및 삭제, 내가 작성한 댓글 조회, 내가 투표한 게시글 조회, 내가 북마크한 게시글 조회 및 북마크 해제 기능을 사용할 수 있어요.<br/>

<img src="https://github.com/CHZZK-Study/Balance-Talk-Frontend/assets/84820008/b0f4a2f6-7940-4fe6-be0a-89af45c77ca6.png" width="700" height="363"/>

### ✅ 투표 기능

밸런스게임 본문 페이지에서 원하는 선택지에 투표할 수 있어요.

투표 후 바로 투표 결과를 확인할 수 있어요.

## 📚 아키텍쳐
```
Balance-Talks-frontend
├── .github                         # Feature, PR, Bug template
├── public                          # public
└── src
    ├── api                         # api 호출 Function
    ├── assets                      # fonts, images, svg resoucres
    ├── components                  # 페이지를 구성하는 components           
    ├── constants                   # api, message, pagination 및 공통적으로 사용되는 constants
    ├── hooks                       # custom hooks                        
    ├── layout                      # 프로젝트 뼈대를 구성하는 layout                         
    ├─- mocks                       # MSW를 활용한 mock data, handler                         
    ├── pages                       # Pages     
    ├── providers                   # Query Provider                                                  
    ├── store                       # Redux-toolkit을 사용한 전역 Store                         
    ├── styles                      # Global Style                         
    ├── types                       # 프로젝트 전반에 걸쳐 사용되는 Type 정의                         
    ├── utils                       # date, validator 관련 util 함수 
```
